# keyboard_teleop
Generic Keyboard Teleop for ROS
#Launch
To run: `rosrun keyboard_teleop keyboard_teleop.py`

#Usage
```
Reading from the keyboard  and Publishing to Twist!
---------------------------
Moving around:
   q    w    e
   a    s    d

anything else : stop

q/z : increase/decrease max speeds by 10%

CTRL-C to quit
```
